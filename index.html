Projeto em Python - Análise de Passageiros do Titanic

Preveja se um passageiro do Titanic sobreviveu com base em informações como idade, gênero, classe socioeconômica, entre outros dados. É um problema de classificação binária onde o aprendizado de máquina é usado para criar um modelo capaz de fazer essa previsão.
