## 🎯 Projeto em Destaque

### 🚀 CobrançaAuto - SaaS em Produção

<p align="center">
  <a href="https://cobrancaauto.com.br" target="_blank">
    <img src="https://img.shields.io/badge/Website-cobrancaauto.com.br-2E9EF7?style=flat-square&logo=google-chrome&logoColor=white" alt="Website"/>
  </a>
  <a href="https://app.cobrancaauto.com.br" target="_blank">
    <img src="https://img.shields.io/badge/App-App.cobrancaauto.com.br-8B5CF6?style=flat-square&logo=react&logoColor=white" alt="App"/>
  </a>
</p>

**O SaaS brasileiro que automatiza cobranças via WhatsApp com pagamento 1-click integrado ao Stripe.**

- 🏗️ **Arquitetura:** Multi-tenant com Docker, MySQL, Redis
- 📱 **Frontend:** React 18 + Vite + TailwindCSS
- 🔧 **Backend:** Laravel 11 + PHP 8.2
- 💳 **Pagamentos:** Stripe Integration
- 📲 **WhatsApp:** Evolution API
